Optimizations

By Nishant Desai


Some optimizations that help:

  1. It was important to ensure that the material and prefabs I used for the stars and constellation lines were not too resource intensive to affect the frame rate drastically.
  2. Using InvokeRepeating instead of update function was also very helpful in increasing performance.
  3. For gameobjects such as the stars, there is no need for the prefab to have shadows. Therfore they can be disabled to save computational load.
  4. Since the dataset has entries for stars that are farther than what users can see from a standpoint, it is useful to have the application render the stars that are within a certain range of the user at a given point in time.
