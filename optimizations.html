Optimizations

By Nishant Desai


Some optimizations that help:

  1. Click on the github link provided above.
  2. Download the project on your machine and unzip it.
  3. In Unity hub go to 'Installs' -> 'Official releases' -> archives and select Unity 2019.X - 2019.2.11 as unity hub link.
  4. Open the project from Unity hub.
  5. In the Assests folder, go to 'module-omicron' -> 'CAVE2' -> 'Scenes' -> 'Cube world example' and drag and drop it into the heirarchy.
  6. Delete the default scene.
  7. Press the play button at the top.
